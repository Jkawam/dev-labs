#  Sobre

O **Dev Playground** é meu laboratório de estudos e experimentação em Desenvolvimento Full Stack.

Este repositório reúne projetos, desafios, exercícios e experimentações desenvolvidos ao longo da minha jornada de aprendizado, colocando em prática conceitos modernos de desenvolvimento Front-end e Back-end.

O principal objetivo é transformar teoria em prática, explorando novas tecnologias, aplicando boas práticas de programação e construindo soluções que contribuam para minha evolução como desenvolvedor.

---

#  Objetivos

- Consolidar conhecimentos em Desenvolvimento Full Stack.
- Praticar lógica de programação e resolução de problemas.
- Desenvolver aplicações Front-end modernas.
- Construir APIs REST utilizando Node.js.
- Explorar bancos de dados relacionais.
- Aprender novas tecnologias continuamente.
- Aplicar boas práticas de desenvolvimento.
- Criar projetos para fortalecer meu portfólio.
